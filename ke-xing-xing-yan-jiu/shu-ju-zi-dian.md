## 定义式

| 符号  | 例子 | 描述 |
| --- | --- | --- |
| = |  |  |
| [,] | a = b + c |  |
| [ I ] | a = [b , c] | 取多个 |
| {} | a = [b I c] | 取一个 |
| m{ }n | 2{b}3 | bb,bbb |
| "" | a = "b" + c + d | 临界编号 |
| （） |  | 可选的 |
| .. | a = 1..9 | 标示一个范围 |
